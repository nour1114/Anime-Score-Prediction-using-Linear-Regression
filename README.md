# Anime Score Prediction using Linear Regression

This project uses Machine Learning with Linear Regression to predict anime scores based on the number of episodes.

## 📌 Features
- Uses Scikit-learn Linear Regression
- Trains a regression model
- Calculates:
  - Coefficient
  - Intercept
  - Mean Squared Error
  - R² Score

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
